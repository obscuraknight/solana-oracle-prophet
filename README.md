# 🐸 SOLANA ORACLE PROPHET

**The Mystic AI Frog Oracle of Solana**

A fun and mystical web app where **Prophet Pepe** gives real-time SOL prices, monthly & yearly forecasts, and spicy predictions about the Solana ecosystem and Pump.fun.

![Prophet Pepe](https://github.com/obscuraknight/solana-oracle-prophet/blob/main/public/images/prophet-pepe.png)

## 🌐 Live Demo

**[👉 Consult the Prophet](https://solana-oracle-prophet.vercel.app/)**

---

## ✨ Features

- **Real-time SOL Price** – Live price + 24h change from CoinGecko
- **Monthly & Yearly Forecasts** – Ask about any month and year (2025–2040)
- **Smart Question Understanding** – Answers even if you phrase it differently
- **Spicy & Mystical Humor** – The Prophet doesn’t give boring answers
- **Dynamic Market Cycle** – Bullish or Bearish animated video based on prediction
- **Beautiful Dark Mystic Design** with Prophet Pepe

---

## 🛠️ Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Real-time data from CoinGecko
- Vercel deployment
- Animated videos + responsive UI
